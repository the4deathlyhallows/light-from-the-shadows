# Testing Checklist

## Query Quality
- [ ] Query compiles
- [ ] Query returns core investigation fields
- [ ] Time range and threshold make sense

## Noise Review
- [ ] Top callers/users reviewed
- [ ] Top devices/hosts reviewed
- [ ] Known admin flows sampled
- [ ] False positives documented

## Operational Fit
- [ ] Severity justified
- [ ] Owner assigned
- [ ] Triage guidance written
- [ ] Promotion criteria set

## Promotion
- [ ] Peer reviewed
- [ ] Test evidence attached
- [ ] Rollback path understood
